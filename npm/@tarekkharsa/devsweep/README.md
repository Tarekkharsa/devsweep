# DevSweep

> Smart process cleanup CLI for developers

Downloads and runs the [DevSweep](https://github.com/TarekKharsa/devsweep) binary.

## Install

```bash
npm install -g @tarekkharsa/devsweep
```

## Usage

```bash
# Scan for dev processes
devsweep scan

# Detect issues
devsweep detect

# Clean up processes
devsweep clean

# Run in watch mode
devsweep watch start

# Show blame report
devsweep blame
```

## Requirements

- Node.js >= 14
- macOS or Linux
- x64 or arm64 architecture
